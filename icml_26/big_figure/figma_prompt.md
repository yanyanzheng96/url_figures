# Figma Design Prompt: VoxelCodeBench Model Comparison Grid

## Overview
Create an **8 rows x 9 columns image grid** comparing voxel generation results from 8 AI models across 9 task categories (3 difficulty levels x 3 task types).

## Grid Structure

### Rows (8 Models - Top to Bottom):
1. Gemini Pro
2. GPT-5
3. GPT-5 Chat
4. GPT-5 Mini
5. Claude 3.5 Sonnet
6. Claude 3 Opus
7. Claude Opus 4
8. Claude Sonnet 4.5

### Columns (9 Tasks):
| Col 1-3 | Col 4-6 | Col 7-9 |
|---------|---------|---------|
| **SYMBOLIC** | **GEOMETRY** | **ARTISTIC** |
| Easy \| Medium \| Hard | Easy \| Medium \| Hard | Easy \| Medium \| Hard |

### Task Details:
- **Symbolic Easy**: Letter "L" shape (symbolic_pattern_3)
- **Symbolic Medium**: Chinese character "光" (artistic_writing_words_18)
- **Symbolic Hard**: "HELLO WORLD" text (artistic_writing_sentences_25)
- **Geometry Easy**: Brown cone (symbolic_single_shape_2)
- **Geometry Medium**: Twisted Cube (geometric_parametric)
- **Geometry Hard**: Wave Surface (geometric_surface)
- **Artistic Easy**: Starfish (artistic_animal_29)
- **Artistic Medium**: Bar Stool with Footrest (artistic_object_99)
- **Artistic Hard**: Rooftop Penthouse (artistic_housing_57)

## Layout Specifications

### Image Cells:
- Size: **810 x 810 px** per image
- Gap between images: **8px**
- Border: **1px solid #E0E0E0**

### Column Headers:
- Height: **80px** total (category + difficulty)
- Category row: **SYMBOLIC** | **GEOMETRY** | **ARTISTIC**
  - Background colors: #E3F2FD (blue) | #E8F5E9 (green) | #FFF3E0 (orange)
- Difficulty row: **Easy** | **Medium** | **Hard** (repeat 3x)

### Row Labels (Left):
- Width: **140px**
- Model names vertically centered
- Font: Inter Medium, 14px
- Background: #FAFAFA

### Total Canvas Size:
- Width: ~7,450px (140 + 9×810 + 8×8)
- Height: ~6,600px (80 + 8×810 + 7×8)
